评测 embedding 时，单一指标很容易误导优化方向。

## 离线阶段

离线建议同时观察：

- 语义相似度任务；
- 检索召回任务；
- 分类迁移任务。

```python
def score_report(sim, recall, cls):
    # 简单示例：按业务权重聚合三个指标
    return 0.4 * sim + 0.4 * recall + 0.2 * cls
```

## 在线阶段

线上要配合点击率、停留时长和人工抽检。离线分数高，不代表线上一定更好。
